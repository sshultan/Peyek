# Latihan Validasi Input Python

## SOAL 1 --- Validasi Nama

Buat program yang: - Meminta input nama - Menolak jika mengandung
**angka atau simbol** - Jika valid, tampilkan: - "Nama valid:
`<nama>`{=html}"

------------------------------------------------------------------------

## SOAL 2 --- Validasi Umur

Buat program yang: - Meminta input umur - Menolak jika input **bukan
angka** - Menolak jika umur \< 0 atau \> 120 - Jika valid, tampilkan
umur

------------------------------------------------------------------------

## SOAL 3 --- Username Tanpa Spasi

Buat program yang: - Meminta username - Username **tidak boleh
mengandung spasi** - Jika ada spasi → tampilkan peringatan

------------------------------------------------------------------------

## SOAL 4 --- Password Minimal

Buat program yang: - Meminta password - Password minimal **8
karakter** - Tampilkan valid atau tidak

------------------------------------------------------------------------

## SOAL 5 --- Cek Angka Genap / Ganjil

Buat program yang: - Meminta input angka - Validasi harus angka -
Tampilkan apakah genap atau ganjil

------------------------------------------------------------------------

## SOAL 6 --- Input Ulang Sampai Benar

Buat program yang: - Meminta nama - Jika ada angka → **mengulang input
otomatis** - Berhenti hanya jika nama valid

------------------------------------------------------------------------

## SOAL 7 --- Login Sederhana

Data:

    username benar: admin
    password benar: 12345

Buat program: - Meminta username dan password - Jika salah → tampilkan
"Login gagal" - Jika benar → tampilkan "Login berhasil"

------------------------------------------------------------------------

## SOAL 8 --- Hitung Rata-rata

Buat program: - Meminta **3 angka** - Jika ada input yang bukan angka →
tampilkan error - Tampilkan rata-ratanya

------------------------------------------------------------------------

## SOAL 9 --- Validasi Tidak Boleh Kosong

Buat program: - Meminta input nama - Jika hanya menekan Enter → tolak
input

------------------------------------------------------------------------

## SOAL 10 --- Validasi Campuran (Challenge)

Buat program yang: - Meminta **nama dan umur** - Nama: tidak boleh
angka - Umur: harus angka - Jika salah satu salah → tampilkan error
khusus sesuai kesalahan
