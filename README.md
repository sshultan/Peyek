### Setup Virtual Environment (Recommended)

```bash
# Buat virtual environment
python3 -m venv virtual_env

# Aktifkan virtual environment
source virtual_env/bin/activate  # Linux/Mac
# atau

```

### Menjalankan Script

```bash

python3 calculator.py

```
